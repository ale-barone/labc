Welcome to LatticeABC's documentation!
======================================

This is a simple library designed for lattice data analyis. The code is a work in progress and constantly
updating. The documentation is therefore under construction and incomplete:
the documented feature are the basic ones that are less likely to undergo major changes.
To get started simply clone the repository

.. code-block:: bash

   git clone https://github.com/ale-barone/labc.git

and use the folder 'labc/labc' as a normal python module.
